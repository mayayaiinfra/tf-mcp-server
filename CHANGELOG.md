# Changelog

All notable changes to @thunderfire/mcp-server will be documented in this file.

## [1.0.0] - 2026-02-21

### Added
- **16 MCP Tools** - Full THUNDERFIRE API coverage
- **3 Resources** - Live node list, health, marketplace catalog
- **2 Prompts** - Guided node creation and debugging workflows
- **TypeScript Server** - Native MCP implementation
- **Python Server** - Alternative implementation via uvx
- **Docker Support** - Container deployment option

### Supported Clients
- Claude Desktop (macOS/Windows)
- Cursor IDE
- VS Code (MCP Extension)
- Any MCP-compatible client

### Configuration
- Environment variables: `THUNDERFIRE_API_URL`, `THUNDERFIRE_API_KEY`
- CLI flags: `--api-url`, `--api-key`, `--timeout`, `--debug`
